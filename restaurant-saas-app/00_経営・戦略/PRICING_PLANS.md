# 💰 料金プラン一覧 (MogMog Plans)

このドキュメントは、アプリケーションで提供されている各料金プランの概要です。最新の詳細は [料金ページ](src/app/plans/page.tsx) の実装を常に正とします。

---

## 1. サービス共通事項

- **初期費用**: ¥0 (※HP制作モードの初回制作費を除く)
- **契約期間**: 縛りなし（いつでも解約可能）
- **維持管理費**: 全プラン料金に含まれています
- **ドメイン**: 標準ドメインは無料。取得価格が4,000円を超えるドメインを希望する場合：取得手数料10,000円 ＋ 4,000円超過分の差額実費。また、2年目以降は更新料実費を別途徴収いたします（本条件に同意したものとします）。
- **所有権**: 制作されたWEBページおよびドメインの管理権限は運営側に帰属します。
- **支払遅延**:
  - **1〜7日**: API機能（AI・SNS連携等）・主要ツールの即時停止。※WEB会員の場合HP公開は維持。
  - **8〜14日**: 管理画面へのログイン遮断。※WEB会員の場合HPも非公開化。
  - **15日**: 契約解除・ドメインおよび制作データ権限の喪失。
  - **31日**: 会員資格抹消、全データの完全削除（復旧不可）。
- **年払い特典**:
  - web Light: 1ヶ月分お得
  - その他のプラン: 2ヶ月分お得 (実質17%OFF)

## 4. よくあるご質問 (概要)

- **AI経験がなくても使えますか？**: はい。ボタン一つでAIが適切な返信文や投稿案を自動作成します。
- **解約金や期間の縛りは？**: 一切ありません。いつでも管理画面から解約・変更可能です。
- **HP制作にはどれくらいの期間がかかりますか？**: 必要な情報が揃ってから、最短3営業日でプレビュー公開、1週間程度で納品可能です。
- **パソコンがないのですが、スマホだけで運用できますか？**: はい。全ての機能はスマートフォンから操作可能です。
- **他社HPからの移行は？**: 可能です。ドメインの引き継ぎ設定などもサポートいたします。
- **悪い口コミの通知は？**: 低評価がついた際に即時通知する機能（ライトプラン以上）があります。
- **対応している支払い方法は？**: クレジットカード（Visa / MasterCard / Amex / JCB等）によるStripe決済に対応しています。
- **年払いから月払いへの変更は？**: 契約更新のタイミングでお切り替えが可能です。

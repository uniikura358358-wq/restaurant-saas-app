# P04_POP・メニュー自動作成_枠組み.md

## 概要 (将来提案)

DALL-E 3 や Stable Diffusion 等の画像生成 AI を連携させ、店舗の POP やメニュー表を自動デザインする機能。

## ロジック・ワークフロー (案)

1. **データ提供**: メニュー名、価格、特徴を UI から入力。
2. **デザイン指示**: テンプレート（和風、カフェ風等）を選択。
3. **画像生成**: プロンプトを AI が自動構成し、画像を生成。
4. **編集**: Canvas API 等を用いた簡易的な文字乗せ・位置調整。
5. **出力**: 印刷用 PDF または高解像度 JPG の書き出し。

## 必要となる設定 (予定)

1. **OpenAI API / Replicate API**:
   - クォータ制限の確保。
2. **Firebase Storage**:
   - 生成済みデザインデータの永続化。

## 環境変数・依存項目 (想定)

- `IMAGE_GEN_API_KEY`: 画像生成 API。
- Package: `canvas` (将来追加), `lucide-react` (v0.563.0)

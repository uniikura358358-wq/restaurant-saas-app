# 01_RULES.md（LINE Growth Engine 運用ルール）

## 1. 禁止・抑止事項

- **煽り配信の抑止**: 誇大広告や誤認を招く表現、強引なクーポン連打を抑止するフィルター/警告を設ける。
- **夜間配信制限**: 原則として21時〜翌日9時の配信は推奨しない。

## 2. 配信頻度（初期プリセット）

- **上限**: 原則として週1回（月4回）。
- **クーポン**: 月2回までを推奨値とする。

## 3. オプトアウト（配信停止）

- 店舗側の責務として、配信メッセージ内またはリッチメニュー等に常に配信停止/ブロックの案内を設ける設計とする。

## 4. トークン管理

- LINE連携に必要なアクセストークンは、最高レベルのセキュリティ（暗号化、定期ローテーション、アクセス制限）で管理する。

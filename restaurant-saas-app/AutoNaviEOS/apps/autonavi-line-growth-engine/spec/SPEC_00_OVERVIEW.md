# SPEC_00_OVERVIEW.md（目的/KPI/対象業態）

## 1. 目的

LINE公式アカウントの運用を仕組み化し、店舗スタッフの負担を増やさずにリピート率と顧客ロイヤリティを向上させる。

## 2. 主要KPI

- **ブロック率**: 配信後のブロック発生率の低減。
- **リピート来店率**: LINE配信をきっかけとした来店の増加。
- **運用継続率**: 配信のネタ切れによる運用の停止を防ぐ。

## 3. 対象

- 公式LINEを開設済みだが、運用が滞っている（または配信ネタが尽きている）店舗。
- Premiumプランの付加価値を最大限活用したい店舗。

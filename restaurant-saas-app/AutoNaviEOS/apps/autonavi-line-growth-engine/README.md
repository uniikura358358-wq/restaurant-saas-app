# AutoNavi LINE Growth Engine

## 1. 概要

LINE公式アカウントの運用を最適化し、再来店や予約を促進するための「運用自動化エンジン」。
頻度制御、カレンダー生成、下書き自動作成、週次レポートを通じて、店舗のLINE運用を仕組み化します。

## 2. コアバリュー

- **配信のやりすぎ防止**: 頻度ガードによりブロック率を低下させる。
- **ネタ切れ解消**: テンプレートとカレンダーにより配信テーマを自動提案。
- **承認するだけ**: 店舗側の作業は「提案された配信案を承認するだけ」に寄せる。

## 3. フォルダ構成

- `docs/`: 専用の憲法・ルール・決定ログ
- `spec/`: 仕様書一式
- `data/`: データ設計
- `ui/`: 画面設計
- `qa/`: テスト観点・受入条件
- `integrations/`: LINE連携手順・ポリシー
- `ops/`: 運用テンプレ・文例
- `legal/`: 配信規約・オプトアウト関連

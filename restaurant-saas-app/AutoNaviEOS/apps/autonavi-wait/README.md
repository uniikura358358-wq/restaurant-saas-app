# AutoNavi Wait

## 1. 概要

「順番待ち」管理を最速で効率化する店舗向けアプリケーション。
ProプランではWeb通知、PremiumプランではLINE通知と予約ブースト機能を提供します。

## 2. コアバリュー

- **最短導入セット (MVP)**: 複雑な設定なしで、その日から順番待ち管理を開始できる。
- **直感的な操作**: 店舗スタッフが迷わず「次へ」進めるUI設計。
- **拡張性**: 将来の機能追加（LINE連携、予約連携）を想定したデータ構造。

## 3. フォルダ構成

- `spec/`: 順番待ち仕様書一式
- `data/`: データ設計・テーブル定義
- `ui/`: 画面設計・ワイヤーフレーム
- `qa/`: テスト観点・受入条件
- `api/`: API設計書
- `legal/`: 利用規約/プライバシー関連
- `deploy/`: デプロイ手順・環境変数

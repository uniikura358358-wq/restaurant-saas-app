# SPEC_02_STATES.md（状態遷移：WAITING/CALLED…）

## 1. チケットの状態遷移

- **WAITING (待機中)**: 受付完了直後。
- **CALLED (呼出中)**: 店舗スタッフが呼び出しを実行した状態。
- **GUIDED (案内済)**: 顧客が来店し、案内が完了した状態（完了）。
- **SKIPPED (スキップ)**: 呼び出しに反応がなく、次の方へ移った状態。
- **CANCELED (キャンセル)**: 顧客自ら、または店舗が取り消した状態。

## 2. 遷移トリガー

- **受付登録** -> `WAITING`
- **呼出ボタン押下** -> `CALLED`
- **案内完了ボタン押下** -> `GUIDED`
- **タイムアウト/スキップボタン** -> `SKIPPED`
- **取消ボタン** -> `CANCELED`

# SPEC_01_USERFLOW.md（客/店フロー）

## 1. 顧客フロー

1. **予約時**: 予約ページで日時・人数・連絡先を入力し、キャンセルポリシーに同意して予約。
2. **前日**: Confirm（確認）通知（メール/LINE）を受信。
3. **確認操作**: 通知内のリンクからワンタップで「来店確定」または「キャンセル」を選択。
4. **完了**: 確定済みの場合は当日来店。

## 2. 店舗フロー

1. **予約登録**: 手入力またはCSVで予約データをインポート。
2. **自動送信**: システムが設定に基づき確認通知を自動送信。
3. **状況確認**: 予約一覧で「確認済」「未回答」「キャンセル」のステータスを監視。
4. **個別対応**: 「未回答」の顧客に対し、必要に応じて電話等の追撃を行う。

# AutoNavi 予約ブースト

## 1. 概要

予約の取りこぼしを防ぎ、直前キャンセルや無断キャンセル（ノーショー）による損失を低減するための予約管理支援システム。
「Confirm（ワンタップ確認）」を中心に、店舗の負担を最小限に抑えた運用を実現します。

## 2. コアバリュー

- **キャンセル損失の低減**: 前日の自動確認により、忘れや重複を防ぐ。
- **店舗負担の最小化**: 設定はプリセット中心で、日々の運用は自動化。
- **透明性**: キャンセルポリシーの明確な提示と同意取得。

## 3. フォルダ構成

- `docs/`: 予約ブースト専用の憲法・ルール・決定ログ
- `spec/`: 仕様書一式
- `data/`: データ設計
- `ui/`: 画面設計
- `qa/`: テスト観点・受入条件
- `ops/`: 運用テンプレ（通知・ポリシー）
- `legal/`: 利用規約/プライバシー関連

# AI原価・回数制限 ストレステスト検証報告書 (2026-02-18)

各プランの「回数上限」まで100%使い切った場合の、理論上の最大原価をシミュレーションした結果です。

## 1. 算出前提 (最悪ケース想定)

- **Gemini 3 Pro (Text)**: 1回あたり **約0.86円** (Input 1.5k / Output 1k トークン想定)
- **Imagen 3 (Image)**: 1回あたり **4.5円** 固定
- **為替**: 150円/$ 固定

---

## 2. プラン別 シミュレーション結果

### A. Standard プラン (予算上限: 600円)

| 項目 | 回数制限 | 単価 | 合計費用 |
| :--- | :--- | :--- | :--- |
| テキスト生成 | 500回 | 0.86円 | 430円 |
| 画像生成 | 50枚 | 4.5円 | 225円 |
| **合計 (最大)** | - | - | **655円** |

> [!WARNING]
> **判定: 予算オーバーのリスクあり**
> 最大消費時（655円）が予算上限（600円）を **55円** 上回っています。
> 現状の `ai-quota.ts` の「二重ガード」により、予算が先に尽きた時点で回数が残っていても停止しますが、ユーザー体験としては「500回できるはずなのに430回で止まった」となる可能性があります。

### B. Pro プラン (予算上限: 1,800円)

| 項目 | 回数制限 | 単価 | 合計費用 |
| :--- | :--- | :--- | :--- |
| テキスト生成 | 1,000回 | 0.86円 | 860円 |
| 画像生成 | 200枚 | 4.5円 | 900円 |
| **合計 (最大)** | - | - | **1,760円** |

> [!NOTE]
> **判定: 安全**
> 予算上限（1,800円）に対し、フル使用で **1,760円**。約40円のバッファがあり、理想的な設定です。

### C. Pro Premium プラン (予算上限: 3,500円)

| 項目 | 回数制限 | 単価 | 合計費用 |
| :--- | :--- | :--- | :--- |
| テキスト生成 | 2,000回 | 0.86円 | 1,720円 |
| 画像生成 | 450枚 | 4.5円 | 2,025円 |
| **合計 (最大)** | - | - | **3,745円** |

> [!CAUTION]
> **判定: 予算オーバーのリスク大**
> 最大消費時（3,745円）が予算上限（3,500円）を **245円** 上回っています。
> 回数制限をすべて使い切る前に、金額ベースのガードレールが作動します。

---

## 3. 対策案

この「回数と予算の乖離」を解消するため、以下のいずれかの調整を推奨します。

### 案①：予算上限を微増させる (推奨)

回数制限を「正」とし、原価がわずかに上振れても回数分は保証する。

- **Standard**: 600円 → **700円**
- **Pro Premium**: 3,500円 → **3,800円**

### 案②：回数制限を微減させる

予算を「絶対死守」とし、回数を調整する。

- **Standard**: 500回 → **430回**
- **Pro Premium**: 2,000回 → **1,700回**

### 案③：二重ガードをそのまま運用する (現状)

「回数」と「予算（円）」のどちらか先に達した方で止める。
原価高騰（為替やThinking Modeの使いすぎ）に対する防御力が最も高いですが、ユーザーへの説明が「回数制限 または 利用枠の超過」と少し複雑になります。

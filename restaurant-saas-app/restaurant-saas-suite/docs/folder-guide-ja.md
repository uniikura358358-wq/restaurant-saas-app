# フォルダの意味（日本語ガイド）

## restaurant-saas-suite
飲食店向けSaaS全体をまとめる親フォルダ。

## google-review-reply
Google口コミ返信機能の作業フォルダ。
例：返信文生成、返信ルール、テンプレート、テストデータ

## instagram-post
Instagram半自動投稿機能の作業フォルダ。
例：投稿文案、ハッシュタグ、画像候補、投稿フロー

## ai-secretary
AI秘書機能の作業フォルダ。
例：領収書読取、質問応答、事務作業支援、タスク補助

## ai-pop-menu
AI POP/メニュー作成機能の作業フォルダ。
例：POPテンプレ、メニュー案、求人POP、販促デザイン案

## office-work-efficiency
事務作業効率化機能の作業フォルダ。
例：売上管理、経費管理、帳簿管理、業務フロー改善

## shared-assets
共通素材フォルダ（全機能で使うもの）。
例：ロゴ、共通テンプレ、アイコン、サンプル画像

## docs
仕様書メモルール計画書を置くフォルダ。
例：要件定義、料金プラン、ロードマップ、運用ルール

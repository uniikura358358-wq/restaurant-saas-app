# プロジェクト概要（日本語）

## 目的
飲食店向けのサブスク型SaaSを開発し、月次収益化を目指す。

## 初期構成（機能）
1. Google口コミ返信
2. Instagram半自動投稿
3. AI秘書
4. AI POP/メニュー作成
5. 事務作業効率化

## 料金プラン（案）
- Standard: 3980円
- Pro: 9800円
- Pro Premium: 14800円

## 作業ルール（初期）
- なるべく親フォルダ配下に素材資料を集約
- 共通素材は shared-assets に保存
- 仕様やルールは docs に記録
- 変更履歴はメモ化して再利用しやすくする

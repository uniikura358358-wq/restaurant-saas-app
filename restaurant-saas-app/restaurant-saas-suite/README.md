# restaurant-saas-suite

## 概要
飲食店向けSaaS機能をまとめたプロジェクトフォルダです。

## 構成
- google-review-reply/      : Google口コミ返信
- instagram-post/           : Instagram投稿補助
- ai-secretary/             : AI秘書
- ai-pop-menu/              : AI POP/メニュー作成
- office-work-efficiency/   : 事務作業効率化
- shared-assets/            : 共通素材
- docs/                     : 資料メモ

## 運用ルール（初期）
- できるだけ親フォルダ配下で作業する（許可ボタン削減対策）
- 共通素材は shared-assets に集約
- 仕様ルールは docs に日本語で記録

# 🏷 タグ定義一覧表（日本語ガイド）

このプロジェクトで使用している英語タグの「日本語での意味」と「役割」をまとめています。

## 1. 状態・進捗（Status/Idea）
| タグ（英語） | 日本語名 | 意味・用途 |
| :--- | :--- | :--- |
| `#status/todo` | **進行中** | 現在作業を進めているプロジェクト |
| `#status/new` | **新規** | 未着手のタスクや新しく出たアイディア |
| `#idea` | **アイデア** | 構想段階のメモ |
| `#project` | **プロジェクト** | 個別のWEB商品開発全体を指すタグ |

## 2. 商品・プロダクト（Product）
| タグ（英語） | 日本語名 | 内容 |
| :--- | :--- | :--- |
| `#product/google-review` | **Google口コミ** | 商品1：口コミ自動返信ツール |
| `#product/insta-auto` | **インスタ投稿** | 商品2：Instagram半自動投稿ツール |
| `#product/ai-secretary` | **AIお助け秘書** | 商品3：事務作業効率化ツール |
| `#product/pop-creator` | **POP作成** | 商品4：AIによるメニュー・POP作成ツール |

## 3. カテゴリ・戦略（Category/Biz）
| タグ（英語） | 日本語名 | 意味・用途 |
| :--- | :--- | :--- |
| `#category/feature` | **機能開発** | プログラムの実装や機能追加に関するノート |
| `#category/marketing` | **集客・マーケ** | 記事作成、広告、SNS運用などの戦略 |
| `#category/new-biz` | **新規事業** | 2サイト目以降の多角経営に関する調査 |
| `#biz/pricing` | **料金設計** | ライト(3980円)〜プレミアム(12800円)の検討 |
| `#biz/scaling` | **事業拡大** | 利益最大化やシステムの横展開アイディア |
| `#category/shared-module` | **共通部品** | 各ツールで使い回す共通のコードや部品 |

---

## 🚀 運用上の注意
- **小文字統一**: `#status/todo` のように、すべて小文字で入力してください。
- **タグの組み合わせ**: 「状態」＋「商品」＋「カテゴリ」をセットで付けるとダッシュボードが見やすくなります。
